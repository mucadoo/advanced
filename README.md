# Advanced Agency

## Description
Advanced Agency is a web design agency specializing in website creation. This project serves as the agency's official website, showcasing its services, team portfolios, and contact information.

## Features
*   **Home**: Welcome section.
*   **About Us**: Information about the agency's academic origins and specialization.
*   **Services**: Displays academic projects developed by the group, including "Hotsite Home Theater Philips" and "Marcenaria Eskalla Site".
*   **Portfolios**: Individual portfolios for each team member (Julio, Samuel, Vantuir, Weber, Wellington), including their resumes and personal works.
*   **Contact**: A contact form for inquiries and additional contact information, including social media links.
*   **Responsive Design**: (Assumed, common practice for web agencies)
*   **Image Gallery**: Utilizes a lightbox effect for portfolio images.

## Technologies Used
*   HTML5
*   CSS3
*   JavaScript (jQuery, Cufon, custom scripts for scrolling, forms, and image handling)

## Installation and Setup
This is a static website. To view it:
1.  Clone the repository to your local machine.
2.  Open the `index.html` file in your web browser.
